公司简介

四川省鑫欧顿商务服务有限公司成立于2011年，坐落于著有“天府之国”之称的四川省成都市，是一家集酒店、保健、浴足为一体的大型综合性连锁企业。公司拥有独立的自主商标和知识产权，负责“鑫欧顿”“伯爵丽都”两大品牌的推广、运营、管理，已发展成中国养生行业最具潜力的加盟连锁机构。经过短短5年时间的跨越式发展，公司目前已经拥有直营店与加盟店近40家，总投资金额达1.8亿元，实现年产值3亿元，现有员工2000余人。公司预计2018年底打造100家实体店面，覆盖全国各省会城市，实现真正意义的全国连锁；2019年底分别成立陕西分公司，河北分公司，北京分公司，山东分公司，上海分公司，广东分公司，福建分公司，实现市场战略部署；2020年底整合养生行业资源，实现融资上市的发展规划，让企业正式走上产业化、多元化、规模化的发展道路。更多...
